# flask-socketio-example

### Create an environment
`$python3 -m venv venv`


### Activate virtualenv
`$source flask-socketio-example/venv/bin/active`


### Install project dependencies
`$pip install -r requirements.txt`


### Run the app:
`$python app.py`


### Open URL on web browser:
`http://127.0.0.1:5000`
